# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| < 1.2   | :x:                |

Only the latest release is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via [GitHub Security Advisories](https://github.com/M-Igashi/region-proxy/security/advisories/new).

Please do **not** open a public issue for security vulnerabilities.

You can expect an initial response within 7 days. If the vulnerability is accepted, a fix will be released as soon as possible.
